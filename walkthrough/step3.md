# Try it

1. Open any git repository in VS Code
2. Make a change and stage it with `git add`
3. Click the **$(heart)** icon on the top right of the ide

Commitr will read your diff and run `git commit -m "..."` in your terminal automatically.

**Keyboard shortcut:** `Ctrl+Shift+Alt+C` (or `Cmd+Shift+Alt+C` on Mac)
